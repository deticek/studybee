\# StudyBee



Aplikacija za izboljšanje fokusa in produktivnosti z uporabo Pomodoro tehnike.



\## 🧠 Opis

Ta aplikacija omogoča uporabnikom učinkovito upravljanje časa z intervalnim delom (Pomodoro metoda), spremljanje napredka ter izboljšanje koncentracije.



\## ✨ Funkcionalnosti

\- Pomodoro timer

\- Zgodovina sej

\- Statistika in streak

\- Prilagoditve zvokov in izgleda (v razvoju)
